paso a paso para la instalación del "Collection" a utilizar para las pruebas del servicio:
1. descargar Postman https://www.postman.com/downloads/ 
Este identifica el sistema operativo para su correcta descarga.
2. crea un usuario en Postman que permitira por la web o aplicacipon usar y sincronizar los workspaces usados
3. descargar el ZIP y descomprimir el archivo "Prueba_Api".
4. en Menú desplegar File -> import -> escoger el archivo (Prueba_Api.postman_collection.json) -> importar
5. el archivo se abrirá automaticamente, al seleccionar "Prueba_Api" se abrira una nueva ventana donde encontrarás el icono de Play, lo que permitira correr las pruebas.
6. adentro encontrará documentación sobre la ejecución de cada prueba como soporte de ayuda.
